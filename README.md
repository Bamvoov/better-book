## üì¶ Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

Ensure you have Node.js installed (Version 16 or higher is recommended).

You can check your version by running:

```
node -v
```

### Installation

1. **Clone the repository**
    
    ```
    git clone [https://github.com/Bamvoov/better-book.git](https://github.com/Bamvoov/better-book.git)
    cd better-book
    ```
    
2. Install dependencies
    
    This step downloads all required libraries (fixing the "Vite not recognized" error).
    
    ```
    npm install
    ```
    
3. **Start the development server**
    
    ```
    npm run dev
    ```
    
4. Open your browser
    
    Navigate to the URL shown in your terminal (usually http://localhost:5173).
